# pitch.ideas
Pitch your business idea and find investors easily

Currently using a Flask server for simplicity (right now its just serving the files).

## Usage

Requires Python installed.

```bash
git clone https://github.com/I-had-a-bad-idea/pitch.ideas.git
cd pitch.ideas
pip install -r requirements.txt
python server.py
```

Then open `http://localhost:4000`
